Lyrics/Poetry Cloze Generator (LPCG) is an add-on for Anki to help with studying long passages of verbatim text, like poetry or song lyrics. The documentation contains information on recommended study techniques and how to use the add-on, and can be read here. The latest version is LPCG 1.3.0. For details on changes, please see the changelog. Support Before asking questions, please take a look at the documentation. If you still have questions, you can email me at anki@sorenbjornstad.com. Please do not post reviews asking questions or reporting bugs, as I have no way to respond to them. If you have found a bug or wish to suggest an improvement, you can post on the issue tracker or submit a pull request at GitHub.
